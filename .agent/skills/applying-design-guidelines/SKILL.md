# Applying Design Guidelines

Ensures all UI components follow the design standards defined in the project.

## When to use this skill

- Generating UI layouts
- Creating dashboards
- Updating UI styles

## Workflow

- [ ] Load design guidelines
- [ ] Validate UI components
- [ ] Apply layout rules
- [ ] Apply spacing and typography

## Instructions

The UI must follow the rules defined in:

```
DesignGuidelines/
```

Guidelines include:

- layout structure
- color system
- typography scale
- component spacing
- animation patterns

Agents must not invent UI styles outside these guidelines.
